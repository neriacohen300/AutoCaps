# 🎙️ AutoCaps - כתוביות אוטומטיות בעברית ל-Premiere Pro ו-DaVinci Resolve

**AutoCaps** הוא תוסף מקומי לחלוטין (100% Offline) המאפשר יצירת כתוביות מדויקות בעברית תוך שניות ישירות לתוך ציר הזמן (Timeline) של תוכנות העריכה המובילות: **Adobe Premiere Pro** ו-**DaVinci Resolve**. 

הפרויקט מבוסס על מנוע `faster-whisper` המהיר ועל מודלי השפה העבריים המתקדמים של **Ivrit-AI** (`ivrit-turbo` / `ivrit-large`), ומספק חוויית עבודה חלקה, ללא צורך בחיבור לאינטרנט וללא עלויות ענן!

---

## ✨ תכונות עיקריות

* 🔒 **מקומי לחלוטין (Privacy First):** כל העיבוד והתמלול נעשים ישירות על המחשב שלכם. שום קובץ קול או וידאו לא נשלח לשרתים חיצוניים.
* ⚡ **מהירות שיא והאצת חומרה:** שימוש במנוע `faster-whisper` המותאם לעבודה מהירה במיוחד על מעבדי CPU ותומך באופן מלא בהאצת כרטיסי מסך של NVIDIA (באמצעות CUDA).
* 🎯 **דיוק פנומנלי בעברית ובאנגלית:** תמיכה במודל `ivrit-turbo` המהיר במיוחד או במודל `ivrit-large` למקסימום דיוק, לצד זיהוי שפה אוטומטי.
* 🖥️ **תמיכה בשתי התוכנות המובילות:**
  * **Adobe Premiere Pro:** הרחבה מעוצבת (CEP Extension) המאפשרת יצוא אודיו מהיר, תמלול וייבוא אוטומטי של קובץ הכתוביות (SRT) ישירות לטיימליין.
  * **DaVinci Resolve:** סקריפט כלי עזר (Lua Utility script) מובנה למערכת ה-Fusion, הפועל בצורה אוטומטית מלאה בגרסת ה-Studio (ובעזרת חצי-אוטומציה נוחה בגרסת ה-Free).
* 🛠️ **התאמה אישית של מבנה הכתוביות:** שליטה מלאה על כמות המילים המקסימלית בשורה, כמות השורות המקסימלית בכתובית, מחיקת סימני פיסוק, חלוקת משפטים חכמה (לכתוביות קצרות וקצביות בסגנון TikTok/Reels) ובחירת טווח (כל הטיימליין או בין נקודות In/Out בלבד).

## 🚀 הוראות התקנה

### דרך א': התקנה אוטומטית ומהירה (Windows)
1. הורידו את קובץ ה-`ZIP` של הגרסה האחרונה מדף ה-Releases.
2. חלצו את תוכן קובץ ה-`ZIP` לתיקייה זמנית.
3. קליק ימני על הקובץ `install_premiere` או `install_resolve תלוי מה אתם רוצים

**הסקריפט יבצע עבורכם את כל הפעולות אוטומטית:**
* התקנת תוסף הפרמייר לתיקיית ה-CEP.
* התקנת סקריפט הדאווינצ'י לתיקיית ה-Utility Scripts של Resolve.
* הגדרת מנוע ה-Backend המשותף בתיקיית ה-AppData של המשתמש.
* זיהוי כרטיס מסך NVIDIA והורדה אוטומטית של ספריות ה-CUDA הדרושות (במידת הצורך).

## 🛠️ כיצד להשתמש?
### שימוש ב-Premiere Pro:
1. פתחו את Premiere Pro ואת הפרויקט שלכם.
2. בתפריט העליון, כנסו אל: Window ◄ Extensions ◄ AutoCaps.
3. בחרו את שפת התמלול הרצויה, את הטווח בטיימליין (הכל או רק In/Out) ואת כמות המילים בשורה.
4. לחצו על כפתור יצירת כתוביות.
5. התוסף ייצא אוטומטית את האודיו, יריץ את התמלול ברקע, וייבא את כתוביות ה-SRT ישירות למסלול הכתוביות של פרמייר!


### שימוש ב-DaVinci Resolve:
1. פתחו את DaVinci Resolve ואת הטיים-ליין שלכם.
2. בתפריט העליון, עברו אל: Workspace ◄ Scripts ◄ AutoCaps .
הגדירו את הפרמטרים הרצויים בתוסף ולחצו על ✨ צור כתוביות.3. 
4. בגרסת Resolve Studio: התוסף ייצא את האודיו, יתמלל, וייבא את הכתוביות אוטומטית ישירות לטיים-ליין!
5. בגרסת Resolve Free: התוסף יעביר אתכם לעמוד ה-Deliver לצורך יצוא האודיו באופן ידני. ברגע שהיצוא יסתיים, התמלול יתחיל מיד ויפתח לכם תיקייה המכילה 6. את קובץ ה-SRT. פשוט גררו את ה-SRT מתיקיית ה-Media Pool לציר הזמן שלכם.
